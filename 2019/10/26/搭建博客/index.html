搭建博客教程

  1. 首先安装node.js，使用npm命令
  2. 准备gitHub仓库，命名为 [your github user].github.io
  3. 安装hexo，首先需要更改npm镜像源（这里使用的是淘宝镜像源）
  4. 接着使用npm命令安装hexo
  5. 使用hexo新建一个项目
  6. 然后进入新建的项目目录，使用npm命令安装依赖库及package.json指定的包
  7. 接下来就是配置修改博客的内容了。部署博客主要是用到以下两条命令

    到此，一个简单的博客就算是完成了创建以及部署，只是本地访问,默认启动的端口4000。

以下是更改博客的主题，更改博客的主题主要是先下载安装一个主题到指定的目录，然后配置文件，做好这些就可以清理数据，重新生成静态文件以及启动服务浏览服务器。
